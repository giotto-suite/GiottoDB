# Reconnect a GiottoDB object

Reconnect a GiottoDB object

## Usage

``` r
# S4 method for class 'GiottoDB'
dbReconnect(x)
```

## Arguments

- x:

  A
  [`GiottoDB`](https://giotto-suite.github.io/GiottoDB/reference/GiottoDB-class.html)
  object.

## Value

A
[`GiottoDB`](https://giotto-suite.github.io/GiottoDB/reference/GiottoDB-class.html)
object with a valid top-level database connection.
